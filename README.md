# Quantium Retail Analytics Project

## Overview
This project analyses customer purchasing behaviour and evaluates the effectiveness of trial store layouts.

## Tasks Performed
- Data cleaning and preparation
- Customer segmentation analysis
- Trial vs control store comparison

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- Trial stores showed increased sales performance
- Growth driven by higher customer numbers
- Families and mainstream customers are key segments
